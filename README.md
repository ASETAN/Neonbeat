# NeonBeat - K-POP Release Tracker

K-POPアーティストの最新リリース情報を追跡できるWebアプリケーション。

## 機能

- 📅 **タイムライン**: フォロー中のアーティストの最新リリースを時系列で表示
- 🔍 **探索**: トレンドのリリースを発見
- 👥 **アーティスト管理**: お気に入りのアーティストをフォロー
- 💾 **ローカル保存**: フォロー情報をブラウザに保存（会員登録不要）
- 🎨 **モダンUI**: ダークモード対応のプレミアムデザイン

## 技術スタック

- **フロントエンド**: HTML, CSS, JavaScript（バニラJS）
- **API**: iTunes Search API
- **ストレージ**: LocalStorage
- **デプロイ**: Vercel

## ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/neonbeat.git

# ディレクトリに移動
cd neonbeat

# ブラウザで index.html を開く
# または Live Server を使用
```

## デプロイ

このプロジェクトは静的サイトなので、Vercel、Netlify、GitHub Pagesなどで簡単にデプロイできます。

### Vercel（推奨）

1. [Vercel](https://vercel.com)でGitHubアカウントでログイン
2. 「Import Project」からこのリポジトリを選択
3. デプロイ完了！
